# 🚀 Dashboard 主页使用指南

> 全面升级的 Obsidian 个人工作台

## 📋 功能概览

| 模块 | 功能 | 使用插件 |
|:---|:---|:---|
| 今日概览 | 日历、时间追踪 | Calendar, Focus Time |
| 今日任务 | 待办任务管理 | Tasks |
| 项目看板 | 项目状态追踪 | Dataview |
| 学习追踪 | 技术栈进度、学习统计 | Dataview |
| 快速入口 | 常用链接导航 | - |

## 🎯 快速开始

### 1. 设置首页（可选）

如果你安装了 **homepage** 插件：
1. 打开 Obsidian 设置
2. 找到 **第三方插件** → **Homepage**
3. 设置首页为 `Dashboard.md`

### 2. 配置 QuickAdd 宏（推荐）

按照 [[🔧 QuickAdd配置]] 中的步骤：
1. 打开 QuickAdd 设置
2. 添加已创建的宏
3. 设置快捷键

### 3. 开始使用

#### 创建日报
- **已配置**：按 `Ctrl+Shift+Q`
- **手动**：打开 [[Templates/今日日报模板]]

#### 添加任务
- **方式1**：在任何笔记中输入任务语法
- **方式2**（已配置）：按 `Ctrl+Shift+A`
- **参考**：[[⚡ 添加快速任务]]

## 📊 各模块说明

### 今日概览
- **日历**：显示本周视图，点击日期可跳转
- **时间追踪**：显示今日学习时长（需配合 Focus Time）

### 今日任务
使用 Tasks 插件语法：
```markdown
- [ ] 今日待办任务 #today
- [x] 已完成任务
```

### 项目进度
- 自动显示所有 `#项目` 标签的笔记
- 按状态分组（待处理/进行中/已完成）
- 显示进度百分比

### 学习追踪
- **技术栈掌握度**：基于 `#技术栈` 标签的任务完成情况
- **学习统计**：基于 `#日报` 标签的学习时长

## 🏷️ 标签系统

| 标签 | 用途 | 示例 |
|:---|:---|:---|
| `#日报` | 每日学习记录 | 7.Daily 日周记/2026-01-30.md |
| `#项目` | 项目笔记 | 1.Projects 项目/学习前端.md |
| `#技术栈` | 技术学习笔记 | 2.Areas 领域/JavaScript基础.md |
| `#问题解决` | 问题解决方案 | Templates/问题解决方案.md |
| `#today` | 今日任务 | `- [ ] 任务 #today` |
| `#habit` | 习惯任务 | `- [ ] 每日阅读 #habit` |

## 💡 使用技巧

1. **快速导航**：使用命令面板（`Ctrl+P`）搜索笔记
2. **任务管理**：在项目笔记中使用 Tasks 插件语法
3. **数据统计**：Dataview 会自动更新，无需手动维护
4. **备份数据**：使用 obsidian-git 插件自动同步

## 🔧 自定义建议

### 添加新的项目
1. 在 `1.Projects 项目/` 创建新笔记
2. 添加 frontmatter：
```yaml
---
状态: 待处理
截止时间: 2026-02-01
tags:
  - 项目
---
```

### 添加技术栈笔记
1. 在 `2.Areas 领域/` 创建笔记
2. 添加 frontmatter：
```yaml
---
tags:
  - 技术栈
---
```

### 自定义任务查询
修改 Dashboard 中的 tasks 块：
```tasks
not done
# 可以添加自定义过滤条件
path includes "学习前端"
```

## ❓ 常见问题

**Q: 日历不显示日期标记？**
A: 确保日报文件有正确的 frontmatter `date: YYYY-MM-DD`

**Q: 项目进度不更新？**
A: 检查项目笔记是否有 `状态` 和 `进度` 字段

**Q: 学习统计为空？**
A: 确保日报文件包含 `time: X小时` 字段
