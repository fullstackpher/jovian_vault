# 🔧 QuickAdd 宏配置指南

> 在 Obsidian 中配置和使用自定义宏

## 📦 已创建的宏

| 宏名称 | 文件 | 功能 |
|:---|:---|:---|
| 快速添加日报 | `QuickAddMacros/快速添加日报.js` | 一键创建今日学习日报 |
| 快速添加任务 | `QuickAddMacros/快速添加任务.js` | 快速添加各类任务 |
| Callout 工具 | `QuickAddMacros/calloutMaker.js` | 快速插入 Callout |

## ⚙️ 配置步骤

### 步骤 1：打开 QuickAdd 设置

1. 打开 Obsidian 设置
2. 找到 **第三方插件** → **QuickAdd**
3. 点击 **Settings** 进入配置

### 步骤 2：添加宏

1. 点击 **Add Macro**（添加宏）
2. 选择对应的 JavaScript 文件：
   - `快速添加日报.js`
   - `快速添加任务.js`
   - `calloutMaker.js`

### 步骤 3：添加命令（可选）

1. 在 QuickAdd 设置中找到 **Add Choice**
2. 选择刚创建的宏
3. 设置命令名称（如："📝 创建日报"）

## ⌨️ 快捷键设置

### 为宏添加快捷键

1. 打开 Obsidian 设置 → **快捷键**
2. 搜索 QuickAdd 命令
3. 为常用宏设置快捷键：

| 命令 | 建议快捷键 | 功能 |
|:---|:---:|:---|
| 快速添加日报 | `Ctrl+Shift+Q` | 创建今日日报 |
| 快速添加任务 | `Ctrl+Shift+A` | 添加任务 |
| Callout 工具 | `Ctrl+Shift+C` | 插入 Callout |

## 🚀 使用方法

### 创建日报

1. 按 `Ctrl+Shift+Q`（或使用命令面板）
2. 输入今日学习时长（如：4小时）
3. 选择重点学习领域
4. 输入今日完成事项（逗号分隔）
5. 输入明日计划（逗号分隔）
6. 自动创建并打开日报

### 添加任务

1. 按 `Ctrl+Shift+A`
2. 选择任务类型（今日/指定日期/习惯/项目）
3. 输入任务描述
4. 根据类型补充额外信息
5. 自动插入到当前笔记

### 插入 Callout

1. 按 `Ctrl+Shift+C`
2. 选择 Callout 类型
3. 输入内容
4. 自动插入到光标位置
