## Number类型
### Number属性

- Number.MAX_SAFE_INTEGER：JavaScript中最大安全整数(2^53 - 1)
- Number.MIN_SAFE_INTEGER：JavaScript中最小安全整数 -(2^53 - 1)

### Number实例方法

- toString(base)，将数字转成字符串，并且按照base进制进行转化
	- base的范围是2到36，默认情况下是10
	- 注意：如果是直接对一个数字进行操作，需要使用`..`运算符
- `toFixed(digits)`，格式化一个数字，保留digits位的小数
	- digits的范围是0-20（包含）之间

### Number方法补充

- `Number.parseInt(String [,radix])`，将字符串解析成整数
- `Number.parseFloat(String)`，将字符串解析成浮点数

### 更多Number的知识，可以查看MDN文档：

- https://developer.mozilla.org/zh-CN/docs/Web/JavaScript/Reference/Global_Objects/Number

## String类型

### String常见的属性：

- `length`：获取字符串的长度

### 访问字符串的字符

- 通过字符串的索引 `str[0]`
- 通过`str.charAt(position)`方法

> [!danger]+ 它们的区别
> 1. 索引的方式没有找到会返回undefined
> 2. charAt没有找到会返回空字符串

### 字符串的遍历

> [!example]+ for普通遍历

```js
for (var i = 0; i < message.length; i++) {
	console.log(message[i])
}
```

> [!example]+ for..of遍历

```js
for (let c of message) {
	console.log(c)
}
```

> [!danger]+ 注意事项
> 1. 对象是不支持for..of
> 2. String对象内部是将字符串变成了一个可迭代对象

### 修改字符串

> [!tip]+ 字符串的不可变性

- 字符串在定义后是不可以修改的，下面的操作没有意义

```js
var message = 'hello world'
message[2] = 'a'
console.log(message);
```

- 返回一个新的字符串
	- `toLowerCase()`：将所有字符转成小写
	- `toUpperCase()`：将所有字符转成大写

```js
var message1 = message.toLocaleLowerCase()
var message2 = message.toUpperCase()
console.log(message1);
console.log(message2);
```

### 查找字符串

- `indexOf(searchString)`：返回某个指定字符串在其他字符串中首次出现的位置索引
	- 有，返回子串所在索引位置
	- 没有，返回-1
	- 应用：可以用来判断包含关系
- `includes(searchString)`：判断是否包含指定字符串，包含返回true，不包含返回false
- `startsWith(searchString)`：判断是否以指定字符串开头
- `endsWith(searchString)`：判断是否以指定字符串结尾
- `replace(regexp|substr, newSubStr|function)`：替换字符串
	- 查找到对应的字符串，并返回替换后的新字符串
	- 可以传入一个正则表达式来查找，也可以传一个函数来替换


### 获取子字符串

> [!success]+ 获取子字符串

| 方法                    | 选择方式                    | 负值参数       |
| --------------------- | ----------------------- | ---------- |
| slice(start, end)     | 从start到end（不包含end）      | 允许负值       |
| substring(start, end) | 从start到end（不包含end）      | 负值代表0      |
| substr(start, length) | 从start开始获取长度为length的字符串 | 允许start为负数 |
> [!tip]+ 截取顺序都是从左到右，开发中推荐使用slice

### 拼接字符串

- 使用 `+` 拼接字符串
- 使用`concat(str)` 拼接字符串
- 使用**模版字符串**（ES6）拼接字符串

```js
// 使用 + 拼接字符串
var left = 'java'
var right = 'script'
var lang = left + right
console.log(lang);

// 使用 concat 拼接字符串
var lang1 = left.concat(right)
console.log(lang1);

// 使用模板字符串拼接字符串
var lang2 = `${left}${right}`
console.log(lang2);
```

### 删除字符串的首尾空格

- `str.trim()`

### 分隔字符串
