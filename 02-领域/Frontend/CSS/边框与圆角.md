### `border`属性
- `border`属性需要三个要素
- `border: 1px solid red;` 
- 线宽度，线型，线颜色

#### 线型
- solid：实线
- dashed：虚线
- dotted：点状线

#### `border-width`
- 边框线宽

#### `border-style`
- 边框线型

#### `border-color`
- 边框线颜色

### 四个方向的边框

- border-top：上边框
- border-right：右边框
- border-bottom：下边框
- border-left：左边框

> 四个方向边框的三要素小属性
- border-top-width：上边框宽度
- border-top-style：上边框线型
- border-top-color：上边框颜色
- ....


#### 去掉边框
- `border-left: none;`即可去掉左边框，以此类推

### `box-shadow`属性
- `box-shadow: 10px 20px 30px rgba(0,0,0, .4);`
	- `h-shadow`：水平偏移，可以为负数
	- `v-shadow`：垂直偏移，可以为负数
	- `blur`：可选值，模糊半径
	- `spread`：可选值，扩展半径
	- `color`：可选值，阴影的颜色
	- `inset`：内阴影（可选值）

### `border-radius`属性
`border-radius` 是 CSS 中用于设置元素边框圆角的属性，可以让元素的角变成圆形或椭圆形。

#### 基本语法
```css
/* 所有角设置相同值 */
border-radius: 10px;

/* 分别设置四个角 */
border-radius: 10px 20px 30px 40px; /* 左上 右上 右下 左下 */

/* 水平和垂直半径分开设置 */
border-radius: 10px / 5px; /* 水平半径10px / 垂直半径5px */

/* 分别设置四个角的水平和垂直半径 */
border-radius: 10px 20px 30px 40px / 5px 10px 15px 20px;
```

#### 属性值类型
##### 1.长度值（px, em, rem等）
```css
border-radius: 15px;
```

##### 2.百分比值（相对于元素的尺寸）
```css
border-radius: 50%; /* 创建圆形 */
```

##### 3.inherit
- 继承父元素

##### 4.initial
- 恢复默认值

#### 单角设置
可以使用单独的属性分别设置每个角：
```css
border-top-left-radius: 10px;
border-top-right-radius: 20px;
border-bottom-right-radius: 30px;
border-bottom-left-radius: 40px;
```

#### 常见示例
##### 1.基本圆形、椭圆
```css
/* 正方形元素 + 50% = 圆形 */
.square {
  width: 100px;
  height: 100px;
  border-radius: 50%;
}

/* 长方形元素 + 50% = 椭圆形 */
.rectangle {
  width: 200px;
  height: 100px;
  border-radius: 50%;
}
```

##### 2.药丸形状按钮
```css
.pill-button {
  padding: 10px 30px;
  border-radius: 100px; /* 使用足够大的值 */
  /* 或 */
  border-radius: 9999px; /* 常用写法 */
}
```

##### 3.不同组合
```css
/* 左右不对称 */
.card {
  border-radius: 10px 20px;
}

/* 只圆左上和右下 */
.element {
  border-radius: 15px 0 15px 0;
}

/* 现代卡片设计 */
.modern-card {
  border-radius: 12px;
}
```

#### 高级用法
##### 1.创建复杂形状
```css
.droplet {
  width: 100px;
  height: 100px;
  border-radius: 50% 50% 50% 0;
  transform: rotate(-45deg);
}
```

##### 2.响应式圆角
```css
.responsive-element {
  border-radius: clamp(8px, 2vw, 20px);
}
```

##### 3.动画效果
```css
.button {
  border-radius: 4px;
  transition: border-radius 0.3s ease;
}

.button:hover {
  border-radius: 20px;
}
```

#### 重要注意事项

1. **裁剪内容**：`overflow: hidden` 时，内容会被圆角裁剪
    
2. **背景和边框**：圆角会同时影响背景和边框
    
3. **性能**：过度复杂的圆角可能影响渲染性能
    
4. **旧浏览器**：IE8 及以下不支持

#### 实际应用场景

```css
/* 用户头像 */
.avatar {
  width: 60px;
  height: 60px;
  border-radius: 50%;
  object-fit: cover;
}

/* 卡片设计 */
.card {
  border-radius: 12px;
  box-shadow: 0 4px 6px rgba(0,0,0,0.1);
}

/* 标签/徽章 */
.tag {
  padding: 4px 12px;
  border-radius: 16px;
  display: inline-block;
}

/* 输入框 */
.input {
  padding: 10px 15px;
  border-radius: 8px;
  border: 1px solid #ddd;
}
```

#### 浏览器兼容性

- 所有现代浏览器完全支持
- IE9+ 支持标准语法
- 需要前缀的旧版本：

```css
-webkit-border-radius: 10px;
-moz-border-radius: 10px;
border-radius: 10px;
```

`border-radius` 是现代 Web 设计的关键属性之一，能够显著提升界面的视觉美感和用户体验。
