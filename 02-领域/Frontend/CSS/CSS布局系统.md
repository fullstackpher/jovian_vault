## 📚 定义
- CSS知识体系的重中之重
- 早期以table为主（简单）
- 后来以技巧性布局为主（难）
- 现在有flexbox/grid（偏简单）
- 响应式布局是必备知识

## 💡 核心理解
> 常用布局方法
- [[table表格布局]]
- [[float浮动+margin布局]]
- [[inline-block布局]]
- [[flexbox布局]] ⭐推荐
- [[响应式布局]]
- 相关基础：[[盒模型]]（布局尺寸计算）、[[行内元素与块元素]]（元素类型）

## 🎯 应用场景

1. PC端（float+margin布局）
2. 移动端（响应式布局，flexbox布局）


## ⚠️ 注意事项

- 浮动布局注意高度塌陷问题，清除浮动
- 行内快布局注意空白间隙问题


## 🔗 实际应用

- 企业官网
- 移动端App
