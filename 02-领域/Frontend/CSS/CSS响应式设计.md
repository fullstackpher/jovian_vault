## 📚 定义
- 1. **媒体查询**
    
    - 设备类型：`screen`、`print`、`speech`
        
    - 媒体特性：`width`、`height`、`orientation`、`aspect-ratio`
        
    - 断点设计策略
        
2. **响应式单位**
    
    - 相对单位：`em`、`rem`、`vw`、`vh`、`%`
        
    - 视口单位应用场景
        
3. **响应式图片**
    
    - `srcset`、`sizes` 属性
        
    - `<picture>` 元素
