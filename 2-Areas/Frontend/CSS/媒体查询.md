> [!NOTE] 媒体查询
> 媒体查询（Media Queries）是CSS3中的一个重要特性，它允许你根据设备的特性（如视口宽度、高度、分辨率等）来应用不同的样式。这使得响应式网页设计成为可能，可以为不同设备提供最佳的用户体验。
> 
> ### 基本语法
> 
> 媒体查询的基本语法如下：
> 
> ```css
> @media mediatype and $expression$ {
>   /* CSS rules */
> }
> ```
> 
> - `mediatype`：指定媒体类型，常见的有 `all`、`screen`、`print` 等。
> - `expression`：一个或多个条件表达式，用于测试特定的媒体特性。
> 
> ### 常见的媒体特性
> 
> 以下是一些常用的媒体特性：
> 
> - `width` 和 `min-width` / `max-width`：视口宽度。
> - `height` 和 `min-height` / `max-height`：视口高度。
> - `device-width` 和 `device-height`：设备屏幕的宽度和高度。
> - `orientation`：设备的方向，值可以是 `portrait` 或 `landscape`。
> - `aspect-ratio` 和 `device-aspect-ratio`：视口或设备的宽高比。
> - `resolution`：设备的分辨率，单位可以是 `dpi` 或 `dpcm`。
> 
> ### 示例
> 
> #### 1. 根据视口宽度调整样式
> 
> ```css
> /* 当视口宽度小于等于600px时应用这些样式 */
> @media $max-width: 600px$ {
>   body {
>     font-size: 14px;
>   }
> }
> 
> /* 当视口宽度大于600px且小于等于900px时应用这些样式 */
> @media $min-width: 601px$ and $max-width: 900px$ {
>   body {
>     font-size: 16px;
>   }
> }
> 
> /* 当视口宽度大于900px时应用这些样式 */
> @media $min-width: 901px$ {
>   body {
>     font-size: 18px;
>   }
> }
> ```
> 
> #### 2. 根据设备方向调整样式
> 
> ```css
> /* 当设备处于横屏模式时应用这些样式 */
> @media $orientation: landscape$ {
>   .container {
>     flex-direction: row;
>   }
> }
> 
> /* 当设备处于竖屏模式时应用这些样式 */
> @media $orientation: portrait$ {
>   .container {
>     flex-direction: column;
>   }
> }
> ```
> 
> #### 3. 根据设备分辨率调整样式
> 
> ```css
> /* 当设备分辨率为192dpi时应用这些样式 */
> @media $resolution: 192dpi$ {
>   img {
>     width: 50%;
>   }
> }
> 
> /* 当设备分辨率为300dpi及以上时应用这些样式 */
> @media $min-resolution: 300dpi$ {
>   img {
>     width: 75%;
>   }
> }
> ```
> 
> ### 使用场景
> 
> - **响应式设计**：根据不同设备的屏幕尺寸调整布局和样式。
> - **打印样式**：为打印页面定义特定的样式。
> - **高分辨率屏幕**：为高分辨率屏幕（如Retina显示屏）提供更清晰的图像和图标。
> 
> 通过合理使用媒体查询，你可以创建出适应多种设备和屏幕尺寸的网站，从而提升用户体验。


> [!NOTE] 断点设计策略
> 断点设计（Breakpoint Design）是一种在响应式网页设计中常用的技术，它允许网站根据用户的设备屏幕尺寸和分辨率自动调整布局。这种设计方法可以确保网站在不同设备上都能提供良好的用户体验。以下是断点设计的一些关键策略：
> 
> ### 1. **理解用户需求**
>    - 在开始设计之前，首先要了解目标用户群体最常使用的设备类型。这有助于确定哪些断点是最重要的。
>    - 使用数据分析工具来收集关于访问者使用设备的信息，如Google Analytics中的设备报告。
> 
> ### 2. **选择合适的断点**
>    - **基于内容的断点**：根据内容自然适合的宽度来设置断点。例如，如果主要内容区域的最佳显示宽度为800px，则可以将此作为其中一个断点。
>    - **基于设备的断点**：针对特定设备或屏幕尺寸设定断点，比如手机、平板电脑和桌面显示器的标准尺寸。
>    - 常见的断点包括：
>      - 小屏幕手机（< 600px）
>      - 大屏幕手机/小平板（600px - 900px）
>      - 平板电脑（900px - 1200px）
>      - 桌面显示器（> 1200px）
> 
> ### 3. **灵活使用CSS媒体查询**
>    - 利用CSS媒体查询来定义不同的样式规则应用于不同的屏幕尺寸。
>    - 示例代码：
>      ```css
>      @media $max-width: 600px$ {
>        /* 手机端样式 */
>        .container { width: 100%; }
>      }
> 
>      @media $min-width: 601px$ and $max-width: 900px$ {
>        /* 平板端样式 */
>        .container { width: 80%; margin: auto; }
>      }
> 
>      @media $min-width: 901px$ {
>        /* 桌面端样式 */
>        .container { width: 70%; margin: auto; }
>      }
>      ```
> 
> ### 4. **保持一致性和简洁性**
>    - 确保在所有断点下，网站的基本功能和信息传达保持一致。
>    - 避免过多复杂的断点设置，以简化维护工作并提高加载速度。
> 
> ### 5. **测试与优化**
>    - 在多种设备和浏览器上测试你的网站，确保每个断点下的表现都符合预期。
>    - 使用开发者工具模拟不同设备查看效果，并根据反馈进行调整。
> 
> 通过以上策略的应用，可以帮助设计师创建出既美观又实用的响应式网站，无论用户使用何种设备访问，都能获得优秀的浏览体验。
