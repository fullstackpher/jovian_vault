> [!NOTE] 媒体查询
> 媒体查询（Media Queries）是CSS3中的一个重要特性，它允许你根据设备的特性（如视口宽度、高度、分辨率等）来应用不同的样式。这使得响应式网页设计成为可能，可以为不同设备提供最佳的用户体验。
> 
> ### 基本语法
> 
> 媒体查询的基本语法如下：
> 
> ```css
> @media mediatype and $expression$ {
>   /* CSS rules */
> }
> ```
> 
> - `mediatype`：指定媒体类型，常见的有 `all`、`screen`、`print` 等。
> - `expression`：一个或多个条件表达式，用于测试特定的媒体特性。
> 
> ### 常见的媒体特性
> 
> 以下是一些常用的媒体特性：
> 
> - `width` 和 `min-width` / `max-width`：视口宽度。
> - `height` 和 `min-height` / `max-height`：视口高度。
> - `device-width` 和 `device-height`：设备屏幕的宽度和高度。
> - `orientation`：设备的方向，值可以是 `portrait` 或 `landscape`。
> - `aspect-ratio` 和 `device-aspect-ratio`：视口或设备的宽高比。
> - `resolution`：设备的分辨率，单位可以是 `dpi` 或 `dpcm`。
> 
> ### 示例
> 
> #### 1. 根据视口宽度调整样式
> 
> ```css
> /* 当视口宽度小于等于600px时应用这些样式 */
> @media $max-width: 600px$ {
>   body {
>     font-size: 14px;
>   }
> }
> 
> /* 当视口宽度大于600px且小于等于900px时应用这些样式 */
> @media $min-width: 601px$ and $max-width: 900px$ {
>   body {
>     font-size: 16px;
>   }
> }
> 
> /* 当视口宽度大于900px时应用这些样式 */
> @media $min-width: 901px$ {
>   body {
>     font-size: 18px;
>   }
> }
> ```
> 
> #### 2. 根据设备方向调整样式
> 
> ```css
> /* 当设备处于横屏模式时应用这些样式 */
> @media $orientation: landscape$ {
>   .container {
>     flex-direction: row;
>   }
> }
> 
> /* 当设备处于竖屏模式时应用这些样式 */
> @media $orientation: portrait$ {
>   .container {
>     flex-direction: column;
>   }
> }
> ```
> 
> #### 3. 根据设备分辨率调整样式
> 
> ```css
> /* 当设备分辨率为192dpi时应用这些样式 */
> @media $resolution: 192dpi$ {
>   img {
>     width: 50%;
>   }
> }
> 
> /* 当设备分辨率为300dpi及以上时应用这些样式 */
> @media $min-resolution: 300dpi$ {
>   img {
>     width: 75%;
>   }
> }
> ```
> 
> ### 使用场景
> 
> - **响应式设计**：根据不同设备的屏幕尺寸调整布局和样式。
> - **打印样式**：为打印页面定义特定的样式。
> - **高分辨率屏幕**：为高分辨率屏幕（如Retina显示屏）提供更清晰的图像和图标。
> 
> 通过合理使用媒体查询，你可以创建出适应多种设备和屏幕尺寸的网站，从而提升用户体验。
