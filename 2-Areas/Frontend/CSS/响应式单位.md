> [!NOTE] CSS相对单位


视口单位应用场景> 在CSS中，相对单位是相对于其他值来定义的尺寸单位。使用相对单位可以使网页设计更加灵活和响应式，因为这些单位会根据屏幕大小、字体大小等因素自动调整。下面是一些常用的CSS相对单位：
> 
> 1. **em**：相对于当前元素的字体大小。如果未设置，则默认为父元素的字体大小。
>    - 例如，`font-size: 2em;` 表示该元素的字体大小是其父元素字体大小的两倍。
> 
> 2. **rem**（root em）：相对于根元素（即 `<html>` 元素）的字体大小。
>    - 例如，如果 `<html>` 的 `font-size` 设置为 16px，那么 `font-size: 1rem;` 就等于 16px。
> 
> 3. **%**（百分比）：相对于包含块（父元素）的宽度或高度。
>    - 例如，`width: 50%;` 表示该元素的宽度是其父元素宽度的一半。
> 
> 4. **vw**（viewport width）：相对于视口（浏览器窗口）的宽度。1vw 等于视口宽度的 1%。
>    - 例如，`width: 50vw;` 表示该元素的宽度是视口宽度的一半。
> 
> 5. **vh**（viewport height）：相对于视口的高度。1vh 等于视口高度的 1%。
>    - 例如，`height: 100vh;` 表示该元素的高度等于视口的高度。
> 
> 6. **vmin**：相对于视口的最小尺寸（宽度或高度中的较小者）。1vmin 等于视口中较小维度的 1%。
>    - 例如，`font-size: 2vmin;` 表示该元素的字体大小是视口较小维度的 2%。
> 
> 7. **vmax**：相对于视口的最大尺寸（宽度或高度中的较大者）。1vmax 等于视口中较大维度的 1%。
>    - 例如，`font-size: 2vmax;` 表示该元素的字体大小是视口较大维度的 2%。
> 
> ### 示例
> 
> 假设有一个简单的HTML结构：
> 
> ```html
> <!DOCTYPE html>
> <html lang="zh">
> <head>
>     <meta charset="UTF-8">
>     <meta name="viewport" content="width=device-width, initial-scale=1.0">
>     <title>相对单位示例</title>
>     <style>
>         html {
>             font-size: 16px;
>         }
>         .container {
>             width: 80%;
>             margin: 0 auto;
>         }
>         .box {
>             width: 50%;
>             height: 50vh;
>             background-color: lightblue;
>             font-size: 2rem;
>         }
>     </style>
> </head>
> <body>
>     <div class="container">
>         <div class="box">这是一个盒子</div>
>     </div>
> </body>
> </html>
> ```
> 
> 在这个示例中：
> - `.container` 的宽度是其父元素（`<body>`）宽度的 80%。
> - `.box` 的宽度是其父元素（`.container`）宽度的 50%。
> - `.box` 的高度是视口高度的 50%。
> - `.box` 的字体大小是根元素（`<html>`）字体大小的 2 倍。
>
