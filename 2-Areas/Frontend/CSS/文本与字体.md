### 继承性
- 文本相关的属性普遍具有继承性，只需要给祖先标签设置，即可在后代所有标签中生效
	- `color`
	- `font-` 开头的
	- `list-` 开头的
	- `text-` 开头的
	- `line-` 开头的
- 因为文字相关的属性有继承性，所以通常会==设置body标签的字号，颜色、行高等，这样就能当做整个网页的默认样式了==
- 就近原则：
	- 在继承的情况下，选择器权重计算失效，而是就近原则 
	- 示例代码
	```html
	<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
    <style>
        #box1 #box2 #box3 {
            color: red;
        }

        p {
            color: green;
        }
    </style>
</head>
<body>
    <div id="box1" class="box1">
        <div id="box2" class="box2">
            <div id="box3" class="box3">
                <p>我是文字</p>
            </div>
        </div>
    </div>
</body>
</html>
	```
### 字体控制

#### `font-family`
- 设置文本的字体系列
	- 微软雅黑
	- 宋体
	- ...

#### `font-style`
- 设置字体样式
	- normal：正常
	- italic：以斜体显示


#### `font-weight`
- 设置字体的粗细，例如：字体加粗bold
	- normal
	- bold
	- bolder
	- 具体数值（400等同于normal，700等同于bold）


#### `font-size`
- 设置字体的大小（通常用的单位是px像素）
	- smaller
	- larger
	- length
	- %

#### `line-height`
- 设置行间距（行高）可以控制文字行与行之间的距离

### 文本属性
#### `color`
- 设置文本的颜色
- 开发中最常用的是十六进制
	- 预定义的颜色值：red、green、blue...
	- 十六进制：#FF0000
	- RGB代码：rgb(255, 0, 0) 或 rgb(100%, 0%, 0%) 

#### `text-align`
- 设置文本的水平对齐方式
	- left：左对齐
	- right：右对齐
	- center：居中对齐

#### `text-decoration`
- 设置文本的装饰，可以给文本添加下划线，删除线，上划线
	- none：默认，没有装饰线（最常用）
	- underline：下划线（常用，a链接自带下划线）
	- overline：上划线（几乎不用）
	- line-through：删除线（不常用）

#### `text-indent`
- 设置文本的第一行缩进，通常是将段落的首行缩进
	- text-indent: 10px;
		
#### `text-transform`
- 可以指定在一个文本中大写和小写字母之间转换
	- uppercase：转大写
	- lowercase：转小写
	- capitalize：首字母大写
```css
p.uppercase {text-transform:uppercase;}
p.lowercase {text-transform:lowercase;}
p.capitalize {text-transform:capitalize;}
```

#### `text-shadow`
- 设置文本阴影
	- h-shadow：水平阴影的位置
	- v-shadow：垂直阴影的位置
	- blur：模糊的距离
	- color：阴影的颜色
```css
/*语法*/
text-shadow: _h-shadow v-shadow blur color_;

/*示例*/
p {text-shadow: 2px 2px 8px #FF0000;}
```

### 溢出处理
#### `text-overflow`
- 规定了当文本溢出包含元素时所发生的事情
	- clip：剪切文本
	- ellipsis：显示省略号代表被剪切的文本
	- string：使用给定的字符串来代表被剪切的文本
	- 需要配合其他两个属性使用：`white-space: nowrap;`和`overflow: hidden;`
示例：==文字溢出部分用省略显示==
```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
    <style>
        .content {
            width: 170px;
            height: 200px;
            padding: 10px;
            border: 1px solid #ccc;
        }
        .info {
            width: 100%;
            overflow: hidden;
            white-space: nowrap;
            text-overflow: ellipsis; /**溢出部分用省略号显示**/
        }
    </style>
</head>
<body>
    <div class="content">
        <img src="images/xiaomi17.jpg">
        <div class="info">【送膜】 UEU 小米17/MAX手机壳保护套硅胶磨砂全包防摔小米17经典黑 - 不漏标</div>
    </div>
</body>
</html>
```

#### `white-space`
- 用来控制空白字符（空格，换行，Tab）要怎么显示
	- normal：默认值，连续的空白字符会被合并成一个空格，文本在必要时会自动换行
	- nowrap：连续的空白字符会被合并成一个空格，但文本不会换行，除非遇到`<br>` 标签
	- pre：保留所有的空白字符（包括空格，制表符和换行符）类似于HTML中`<pre>`标签
	- pre-wrap：保留所有的空白字符（包括空格，制表符和换行符）文本在必要时会自动换行
	- pre-line：保留换行符，但会合并连续的空白字符为一个空格，文本在必要时会自动换行

- 使用场景：
	- normal：适用于大多数的普通文本内容，确保文本格式整洁且可读
	- nowrap：适用于需要防止文本换行的情况，例如导航栏或表格单元格
	- pre：适用于需要完全保留原始格式的文本，例如代码块或诗歌
	- pre-line：适用于需要保留换行符但不保留额外空白字符的文本
	- pre-wrap：适用于需要保留所有空白符并允许自动换行的文本
