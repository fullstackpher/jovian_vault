### 继承性
- 文本相关的属性普遍具有继承性，只需要给祖先标签设置，即可在后代所有标签中生效
	- color
	- font- 开头的
	- list- 开头的
	- text- 开头的
	- line- 开头的
- 因为文字相关的属性有继承性，所以通常会==设置body标签的字号，颜色、行高等，这样就能当做整个网页的默认样式了==
- 就近原则：
	- 在继承的情况下，选择器权重计算失效，而是就近原则 
	- 示例代码
	```html
	<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
    <style>
        #box1 #box2 #box3 {
            color: red;
        }

        p {
            color: green;
        }
    </style>
</head>
<body>
    <div id="box1" class="box1">
        <div id="box2" class="box2">
            <div id="box3" class="box3">
                <p>我是文字</p>
            </div>
        </div>
    </div>
</body>
</html>
	```
### 字体控制

> font-family
- 设置文本的字体系列
	- 微软雅黑
	- 宋体
	- ...

> font-style
- 设置字体样式
	- normal：正常
	- italic：以斜体显示


> font-weight
- 设置字体的重量，例如：字体加粗bold
	- normal
	- bold
	- bolder


>font-size
- 设置字体的大小
	- smaller
	- larger
	- length
	- %


>line-height
- 设置字体的行高
