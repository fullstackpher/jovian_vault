CSS背景（background）用于设置元素的背景样式，包括颜色、图片、渐变等。以下是核心属性和用法详解：

---

## **一、常用背景属性**

### 1. **背景颜色**  
```css
div {
  background-color: #ff0000;        /* 十六进制 */
  background-color: rgb(255, 0, 0); /* RGB */
  background-color: rgba(255, 0, 0, 0.5); /* RGB + 透明度 */
  background-color: hsl(0, 100%, 50%); /* HSL */
}
```

### 2. **背景图片**  
```css
div {
  background-image: url("image.jpg");
  background-image: linear-gradient(to right, red, blue); /* 渐变背景 */
}
```

### 3. **背景重复**  
```css
div {
  background-repeat: repeat;    /* 默认，重复铺满 */
  background-repeat: no-repeat; /* 不重复 */
  background-repeat: repeat-x;  /* 水平重复 */
  background-repeat: repeat-y;  /* 垂直重复 */
}
```

### 4. **背景定位**  
```css
div {
  background-position: center;        /* 居中 */
  background-position: 10px 20px;     /* 水平10px 垂直20px */
  background-position: right bottom;  /* 右下角 */
}
```

### 5. **背景大小**  
```css
div {
  background-size: cover;      /* 覆盖整个区域，可能裁剪 */
  background-size: contain;    /* 完整显示图片，可能留白 */
  background-size: 100px 200px; /* 指定宽高 */
  background-size: 50% 80%;    /* 相对容器比例 */
}
```

### 6. **背景附着**  
```css
div {
  background-attachment: scroll; /* 默认，随内容滚动 */
  background-attachment: fixed;  /* 固定于视口，实现视差效果 */
}
```

### 7. **背景原点与裁切**  
```css
div {
  background-origin: padding-box; /* 背景从内边距开始（默认） */
  background-clip: content-box;   /* 背景仅在内容区显示 */
}
```

---

## **二、简写属性**
```css
div {
  background: #000 url("image.jpg") center/cover no-repeat fixed;
}
/* 顺序：颜色 图片 位置/大小 重复 附着 */
```

---

## **三、渐变背景**
### 1. **线性渐变**
```css
div {
  background: linear-gradient(to right, red, blue);
  background: linear-gradient(45deg, red 20%, blue 80%);
}
```

### 2. **径向渐变**
```css
div {
  background: radial-gradient(circle, red, yellow);
}
```

### 3. **重复渐变**
```css
div {
  background: repeating-linear-gradient(45deg, red 0px, red 10px, blue 10px, blue 20px);
}
```

---

## **四、多背景图层**
```css
div {
  background: 
    url("layer1.png") top left no-repeat,
    url("layer2.png") bottom right repeat-x,
    linear-gradient(rgba(0,0,0,0.5), transparent);
}
/* 多个背景层叠，最先写的在最上层 */
```

---

## **五、实用技巧**
### 1. **全屏背景**
```css
body {
  background: url("bg.jpg") center/cover no-repeat fixed;
  min-height: 100vh;
}
```

### 2. **背景滤镜**
```css
div {
  background: url("image.jpg");
  filter: blur(5px); /* 模糊背景 */
}
```

### 3. **背景混合模式**
```css
div {
  background: url("image.jpg"), linear-gradient(red, blue);
  background-blend-mode: multiply;
}
```

### 4. **背景滚动视差**
```css
.parallax {
  background: url("bg.jpg") center/cover fixed;
}
```

---

## **六、注意事项**
1. **性能优化**：大图背景使用 `background-size: cover` 可能导致移动端性能问题。
2. **渐变兼容**：旧浏览器需加前缀 `-webkit-`、`-moz-`。
3. **后备颜色**：图片未加载时显示背景色：
   ```css
   div {
     background: #ccc url("image.jpg");
   }
   ```

---

## **七、示例组合**
```css
.card {
  background: 
    linear-gradient(135deg, rgba(255,255,255,0.1) 0%, transparent 50%),
    url("pattern.png") center/auto repeat,
    #2a2a2a;
  background-blend-mode: overlay;
  border-radius: 12px;
}
```

掌握这些背景属性后，可以创建复杂的视觉层次和动态效果！
