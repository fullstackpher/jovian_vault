### `transform`属性
- `transform`属性应用于元素的2D和3D转换，允许将元素旋转，缩放，移动，倾斜等
- 


### `transition`过渡
- transition过渡属性是CSS3浓墨重彩的特性，过渡可以为一个元素在不同样式之间变化自动添加 ”补间动画“
	![](https://jovian-1257905003.cos.ap-shanghai.myqcloud.com/picture/局部截取_20260120_132044.png)
- 兼容性：
	- 过渡从IE10开始兼容，移动端兼容良好
- 曾经网页的动画都是由JS定时器实现的，现在逐步改为了CSS过渡
- 优点：动画更细腻，内存开销小

### `transition`属性
- `transition: width 1s linear 0s;`
- `ransition-property`：什么属性要过渡
	- 所有数值类型的属性，都可以参与过渡，
		- 比如`width`，`height`，`left`, `top`，`border-radius`
	- 背景颜色和文字颜色都可以被过渡
	- 所有变形（包括2D和3D）都能被过渡
	- `all`属性：如果要所有属性都参与过渡，可以写`all`（使用需谨慎）
- `transition-duration`：定义过渡动画持续时长
- `transition-timing-function`：定义过渡的速度曲线
	- `linear`：匀速（线性）
	- `ease`：渐入渐出
	- `ease-in`：渐入
	- `ease-out`：渐出
	- `ease-in-out`：先渐入后渐出
- `transition-delay`：定义过渡效果开始前的延迟时间
	- 值：如0s，1s，也可以是负数，则使动画从中途开始

### 注意事项
- 并非所有属性都可以过渡，只有具有**中间状态**的属性才可以
- 例如：`display: block -> none` 则不能过渡
- 性能优先：过渡对以下属性对性能更友好
	- `transform`（移动，旋转，缩放）
	- `opacity`
	- `filter`（模糊，阴影）
	- 避免过渡会触发[[页面重排]]的属性，如`width`，`height`，`margin`，`top`/`left`等，尤其是在动画元素很多的情况下
- 初始状态必须明确：浏览器需要知道从哪个值开始过渡。确保过渡生效的初始状态已在样式中定义
- 反向过渡：通常，需要在元素的基础状态定义transition，这样在返回初始状态时也会有动画效果，这是一种常见的**往返**动画模式

### 过渡效果实战案例

> [!info] 显示隐藏的标题

```html
<!DOCTYPE html>  
<html lang="en">  
<head>  
    <meta charset="UTF-8">  
    <title>过渡效果实战1</title>  
    <style>  
        *{  
            margin: 0;  
            padding: 0;  
        }  
        .box {  
            width: 1200px;  
            overflow: hidden;  
            margin: 40px auto;  
        }  
  
        .box ul {  
           list-style: none;  
        }  
  
        .box ul li {  
            float: left;  
            width: 380px;  
            height: 210px;  
            margin-right: 20px;  
            position: relative;  
        }  
  
        .box ul li img {  
            width: 380px;  
            height: 210px;  
        }  
  
        .box ul li .info {  
            position: absolute;  
            width: 370px;  
            padding-left: 10px;  
            height: 20px;  
            line-height: 20px;  
            color: #fff;  
            bottom: 0;  
            background: rgba(0,0,0,0.5);  
            opacity: 0;  
            transition: opacity 1s ease 0s;  
        }  
  
        .box ul li:hover .info {  
            opacity: 1;  
        }  
    </style>  
</head>  
<body>  
<div class="box">  
    <ul>  
        <li>  
            <img src="images/0.jpg" alt="">  
            <div class="info">北京的故宫</div>  
        </li>  
        <li>  
            <img src="images/1.jpg" alt="">  
            <div class="info">十七孔桥</div>  
        </li>  
        <li>  
            <img src="images/2.jpg" alt="">  
            <div class="info">国家大剧院</div>  
        </li>  
    </ul>  
</div>  
</body>  
</html>
```
