### `transform`属性
> `transform` 属性是 CSS 中非常强大的一个属性，它允许你对元素进行旋转、缩放、移动（平移）或倾斜等二维或三维变换。这个属性可以用来创建各种视觉效果，并且对于响应式设计和动画制作非常有用。
> 
> ### 基本语法
> 
> ```css
> transform: <transform-function> | none;
> ```
> 
> - `<transform-function>` 是一个或多个变换函数的列表，这些函数可以按顺序应用。
> - `none` 表示不应用任何变换。
> 
> ### 常见的变换函数
> 
> 1. **translate$x, y$**
>    - 平移元素，`x` 和 `y` 分别表示水平和垂直方向上的位移量。
>    - 例如：`transform: translate$50px, 30px$;`
> 
> 2. **scale$x, y$**
>    - 缩放元素，`x` 和 `y` 分别表示水平和垂直方向上的缩放比例。
>    - 例如：`transform: scale$1.5, 1.2$;`
> 
> 3. **rotate$angle$**
>    - 旋转元素，`angle` 表示旋转的角度，单位可以是度 $`deg`$、弧度 $`rad`$、梯度 $`grad`$ 或圈数 $`turn`$。
>    - 例如：`transform: rotate$45deg$;`
> 
> 4. **skew$x-angle, y-angle$**
>    - 倾斜元素，`x-angle` 和 `y-angle` 分别表示水平和垂直方向上的倾斜角度。
>    - 例如：`transform: skew$20deg, 10deg$;`
> 
> 5. **matrix$a, b, c, d, e, f$**
>    - 使用矩阵进行变换，适用于更复杂的变换组合。
>    - 例如：`transform: matrix$1, 0, 0, 1, 50, 30$;`
> 
> 6. **translateX$x$ 和 translateY$y$**
>    - 分别沿 X 轴和 Y 轴平移元素。
>    - 例如：`transform: translateX$50px$;`
> 
> 7. **scaleX$x$ 和 scaleY$y$**
>    - 分别沿 X 轴和 Y 轴缩放元素。
>    - 例如：`transform: scaleX$1.5$;`
> 
> 8. **rotateX$angle$ 和 rotateY$angle$**
>    - 分别绕 X 轴和 Y 轴旋转元素。
>    - 例如：`transform: rotateX$45deg$;`
> 
> 9. **perspective$length$**
>    - 设置透视距离，用于三维变换。
>    - 例如：`transform: perspective$500px$;`
> 
> ### 示例
> 
> 假设我们有一个正方形的 `div`，我们想让它旋转 45 度并放大 1.5 倍：
> 
> ```html
> <div class="box"></div>
> ```
> 
> ```css
> .box {
>   width: 100px;
>   height: 100px;
>   background-color: blue;
>   transform: rotate$45deg$ scale$1.5$;
> }
> ```
> 
> ### 注意事项
> 
> - `transform` 属性不会影响布局，即变换后的元素不会影响其他元素的位置。
> - 可以结合 `transition` 属性来创建平滑的动画效果。
> - 在使用三维变换时，需要确保浏览器支持 CSS3 的三维变换特性。

### `transition`过渡
- transition过渡属性是CSS3浓墨重彩的特性，过渡可以为一个元素在不同样式之间变化自动添加 ”补间动画“
	![](https://jovian-1257905003.cos.ap-shanghai.myqcloud.com/picture/局部截取_20260120_132044.png)
- 兼容性：
	- 过渡从IE10开始兼容，移动端兼容良好
- 曾经网页的动画都是由JS定时器实现的，现在逐步改为了CSS过渡
- 优点：动画更细腻，内存开销小

### `transition`属性
- `transition: width 1s linear 0s;`
- `ransition-property`：什么属性要过渡
	- 所有数值类型的属性，都可以参与过渡，
		- 比如`width`，`height`，`left`, `top`，`border-radius`
	- 背景颜色和文字颜色都可以被过渡
	- 所有变形（包括2D和3D）都能被过渡
	- `all`属性：如果要所有属性都参与过渡，可以写`all`（使用需谨慎）
- `transition-duration`：定义过渡动画持续时长
- `transition-timing-function`：定义过渡的速度曲线
	- `linear`：匀速（线性）
	- `ease`：渐入渐出
	- `ease-in`：渐入
	- `ease-out`：渐出
	- `ease-in-out`：先渐入后渐出
- `transition-delay`：定义过渡效果开始前的延迟时间
	- 值：如0s，1s，也可以是负数，则使动画从中途开始

### 注意事项
- 并非所有属性都可以过渡，只有具有**中间状态**的属性才可以
- 例如：`display: block -> none` 则不能过渡
- 性能优先：过渡对以下属性对性能更友好
	- `transform`（移动，旋转，缩放）
	- `opacity`
	- `filter`（模糊，阴影）
	- 避免过渡会触发[[页面重排]]的属性，如`width`，`height`，`margin`，`top`/`left`等，尤其是在动画元素很多的情况下
- 初始状态必须明确：浏览器需要知道从哪个值开始过渡。确保过渡生效的初始状态已在样式中定义
- 反向过渡：通常，需要在元素的基础状态定义transition，这样在返回初始状态时也会有动画效果，这是一种常见的**往返**动画模式

```html
<!DOCTYPE html>  
<html lang="en">  
<head>  
    <meta charset="UTF-8">  
    <title>过渡效果实战1</title>  
    <style>  
        *{  
            margin: 0;  
            padding: 0;  
        }  
        .box {  
            width: 1200px;  
            overflow: hidden;  
            margin: 40px auto;  
        }  
  
        .box ul {  
           list-style: none;  
        }  
  
        .box ul li {  
            float: left;  
            width: 380px;  
            height: 210px;  
            margin-right: 20px;  
            position: relative;  
        }  
  
        .box ul li img {  
            width: 380px;  
            height: 210px;  
        }  
  
        .box ul li .info {  
            position: absolute;  
            width: 370px;  
            padding-left: 10px;  
            height: 20px;  
            line-height: 20px;  
            color: #fff;  
            bottom: 0;  
            background: rgba(0,0,0,0.5);  
            opacity: 0;  
            transition: opacity 1s ease 0s;  
        }  
  
        .box ul li:hover .info {  
            opacity: 1;  
        }  
    </style>  
</head>  
<body>  
<div class="box">  
    <ul>  
        <li>  
            <img src="images/0.jpg" alt="">
```
