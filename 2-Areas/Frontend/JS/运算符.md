### 运算符和运算元

- 在小学的时候，就学习了各种运算符，比如：加减乘除
- 几乎所有的编程语言都会有各种各样的运算符（也叫操作符，operators）
- 计算机最基本的操作就是执行运算，执行运算时就需要使用运算符来操作：
- JavaScript按照使用场景的不同将运算符分成了很多种类型：
	- 算术运算符/赋值运算符/关系(比较)运算符/逻辑运算符

- 运算元 - **运算符应用的对象**
	- 比如乘法运算：`5 * 2` ，有两个运算元
	- 左运算元 5 和右运算元 2
	- 也叫参数
- 如果一个运算符对应的只有**一个运算元**，那么它是**一元运算符**
	- 比如**一元负号运算符**（`unary negation`)，它的作用是对数字进行正负转换的
- 如果一个运算法拥有两个运算元，那么它是**二元运算符**
	- 比如`2 + 3`

### 算数运算符

- 算术运算符用在数学表达式中，它的使用方式和数学中是一样的
- 算术运算符是对数据进行计算的符号；

| 运算符  | 运算规则   | 范例      | 结果   |
| ---- | ------ | ------- | ---- |
| `+`  | 加法     | 2 + 3   | 5    |
| `+`  | 连接字符串  | '中'+'国' | '中国' |
| `-`  | 减法     | 2 - 3   | -1   |
| `*`  | 乘法     | 2 * 3   | 6    |
| `/`  | 除法     | 5 / 2   | 2.5  |
| `%`  | 取余数    | 5 % 2   | 1    |
| `**` | 幂（ES7） | 2 ** 3  | 8    |

### 赋值运算符

> [!tip] = 其实也是一个运算符，叫做赋值运算符（assignments）

- = 是一个运算符，语句 x = value 将值value写入x然后返回x
- 链式赋值（chaining assignments）
	- `let a = b = c = 123;`
	- 链式赋值从右到左进行计算
	- 从代码可读性的角度来说，不推荐这种写法

>[!tip] 原地修改（Modify-in-place）

- 我们经常需要**对一个变量做运算**，并**将新结果存储在同一个变量中**
- 可以使用`+=`和`*=`这种方式来缩写这种表示
- 所有算术和位运算都有简短的 “**修改并赋值**” 运算符：/=和-=等

| 运算符   | 运算规则  | 范例               | 结果  |
| ----- | ----- | ---------------- | --- |
| `=`   | 赋值    | `a = 5`          | 5   |
| `+=`  | 加后赋值  | `a = 5, a += 2`  | 7   |
| `-=`  | 减后赋值  | `a = 5, a -= 2`  | 3   |
| `*=`  | 乘后赋值  | `a = 5, a *= 2`  | 10  |
| `/=`  | 除后赋值  | `a = 5, a /= 2`  | 2.5 |
| `%=`  | 取余后赋值 | `a = 5, a %= 2`  | 1   |
| `**=` | 幂后赋值  | `a = 5, a **= 2` | 25  |

### 自增和自减
- 对一个数进行**加一**和**减一**是最常见的数学运算符之一
- 自增：`++`将变量加一
- 自减：`--`将变量减一
- 自增自减只能用于变量
	- 将其应用于数值（比如5++）则会报错

 >[!tip] 自增/自减表达式在其他表达式中时
 >前缀形式：先自增/自减，然后返回自增/自减后的值
 >后缀形式：先返回变量当前的值，然后再自增/自减。

### 运算符优先级

下表按从最高到最低的优先级列出JavaScript运算符，具有相同优先级的运算符按从左至右的顺序求值。

|优先级|分类|运算符|结合性|
|---|---|---|---|
|1|后缀运算符|( )、[ ]、->|从左到右|
|2|单目运算符|!、*（指针）、& 、++、--、+（正号）、-（负号）|`从右到左`|
|3|乘法/除法/取余|*（乘号）、/、%|从左到右|
|4|加法/减法|+、-|从左到右|
|5|位移运算符|<<、>>|从左到右|
|6|关系运算符|<、<=、>、>=|从左到右|
|7|相等/不等|==、!=|从左到右|
|8|按位与|&|从左到右|
|9|按位异或|^|从左到右|
|10|按位或|\||从左到右|
|11|逻辑与|&&|从左到右|
|12|逻辑或|\||从左到右|
|13|三目运算符|? :|`从右到左`|
|14|赋值运算符|=、+=、-=、*=、/=、 %=、 >>=、 <<=、&=、^=、\|=|`从右到左`|
|15|逗号运算符|,|从左到右|

### 比较运算符

- 大于 / 小于：a>b，a < b
- 大于等于 / 小于等于：a>= b，a<=b
- 检查两个值的相等：a == b，注意 == 表示相等性检查，而单=表示赋值
- 检查两个值不相等：a != b

>[!tip] 比较运算符的结果都是Boolean类型的

| 运算符 | 运算规则 | 范例     | 结果    |
| --- | ---- | ------ | ----- |
| ==  | 相等   | 4 == 3 | false |
| !=  | 不等于  | 4 != 3 | true  |
| >   | 大于   | 4 > 3  | true  |
| <   | 小于   | 4 < 3  | false |
| >=  | 大于等于 | 4 >= 3 | true  |
| <=  | 小于等于 | 4 <= 3 | false |
> [!tip] === 和 == 的区别

- 普通的相等性检查 == 存在一个问题，它不能区分出0和false，或空字符串和false这类运算
- 1. 严格相等（ === ）：
    - 比较两个值是否相等，且不进行类型转换。
    - **只有当两个操作数的值和类型都相同时，才返回true**。

1. 宽松相等（ == ）：
    - 比较两个值是否相等，但在比较前会进行类型转换（强制类型转换）。
    - **如果两个操作数的类型不同，== 会尝试将它们转换为相同类型，然后再比较值**。

### 三目运算符

> 有时候，我们需要根据一个条件去赋值一个变量

- 比如比较数字大小，获取较大的数字
- 这个时候使用ifelse就会显得过于臃肿，那么三目运算符就很好的解决了这个问题

> 条件运算符：`?`

- 这个运算符通过问号 ？表示
- 有时他被称为三元运算符，因为该运算符有三个操作数（运算元）
- 实际上它是JavaScript中唯一一个拥有这么多操作数的运算符

> [!tip] 使用格式如下：
> var result = condition ? value1 : value2;

### 逻辑运算符

> 逻辑运算符，主要有三个：

- ||（或），&&（与），!（非）
- 它可以将多个表达式或值放到一起，来获取到一个最终的结果

| 运算符  | 运算规则   | 范例              | 结果    |
| ---- | ------ | --------------- | ----- |
| &&   | 与：同时为真 | false && true   | false |
| \|\| | 或：一个为真 | false \|\| true | true  |
| !    | 非：取反   | !false          | true  |
