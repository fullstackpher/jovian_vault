### 程序的执行顺序

>[!tip] 在程序开发中，程序有三种不同的执行方式

- 顺序 - **自上而下**，顺序执行代码
- 分支 - 根据**条件判断**，决定执行代码的**分支**
- 循环 - 让 **特定代码 重复** 执行

### 代码块的理解

- 代码块是多行执行代码的集合，通过花括号`{}`放到了一起
- 在开发中，一行代码很难完成某一个特定的功能，我们会将这些代码放到一个代码块中
- 在JavaScript中，我们可以通过**流程控制语句**来决定如何执行一个代码块
	- 通过会通过一些**关键字**来告知**JS引擎**代码要如何被执行
	- 比如**分支语句**，**循环语句**对应的关键字等

### 分支结构

- 分支结构的代码就是让我们**根据条件**来**决定代码的执行**
- 分支结构也叫**判断结构**或**选择结构**
- 几乎所有的编程语言都有分支结构（C、C++、OC、Java、Python等）

> JavaScript中常见的分支结构有：
- `if`分支结构
- `switch`分支结构

### if分支语句

>if分支结构有三种

- 单分支结构
	- `if...`
- 多分枝结构
	- `if...else...`
	- `if...else if...else...`

#### 单分支结构
- 单分支语句：`if`
	- `if(...)`语句计算括号内的条件表达式，如果计算结果是true，就会执行对应的代码块
```js
// 如果条件成立，那么执行代码块
if(条件判断) {
	// 执行代码块
}
```

>[!tip] 如果代码中只有一行代码，那么`{}`可以省略

> `if(...)`语句会计算圆括号内的表达式，并将计算结果转换为布尔型（Boolean）

- 转换规则和Boolean函数的规则一致
- 数字0，空字符串“”，null，undefined和NaN都会被转为false，这些都是假值（`falsy`）
- 其他值被为true，所以它们被称为真值（`truthy`）
#### 多分支结构
- 多分支语句：`if...else...`
	- `if`语句有时候包含一个可选的`else`块
	- 如果**判断条件不成立**，那么就会执行`else`块的代码
	```js
	// 如果条件成立，那么执行代码块
	// 如果条件不成立，执行另一段代码块
	if (条件判断) {
		// 条件成立，执行代码块
	} else {
		// 条件不成立，执行代码块
	}
	```

- 多分支语句：`if...else if...else...`
	- 有时候我们需要判断多个条件
	- 我们可以通过使用else if子句实现
```js
if(条件1) {
	// 代码块
} else if(条件2) {
	// 代码块
} else if(条件3) {
	// 代码块
} else {
	// 代码块
}
```

### switch语句

> switch是分支结构的一种语句

- 它是通过判断表达式的结果（或变量）是否等于case语句的常量，来执行相应的分支体的
- 与if语句不同的是，switch语句只能做**值的相等判断**，而if语句可以做**值的范围判断**
- switch语法：
	- switch语句有至少一个case代码块和一个可选的default代码块
```js 
switch(变量) {
	case 常量1：
		break;
	case 常量2:
		break;
	default:
	 //default语句
}
```

>[!tip] case穿透问题：
>一条case语句结束后，会自动执行下一个case语句，这种现象被称为case穿透

>[!tip] break关键字
>通过在每个case的代码块后添加break关键字来解决这个case穿透问题

>[!danger] 注意事项：这里的相等是 “严格相等”
>被比较的值必须是相同的类型才能进行匹配

### 认识循环

循环是一种重复运行同一代码的方法

如果是对某一列表进行循环操作，我们通常也叫**遍历**（traversal）或**迭代**（iteration）

在JavaScript中支持三种循环方式：
- while循环
- do...while循环
- for循环

### while循环

>while循环语法如下：

- 当条件成立时，执行代码块
- 当条件不成立时，跳出代码块
```js
while(循环条件) {
	// 循环代码块
}
```

- 如果条件一直成立，那么就会产生死循环
- 此时必须通过关闭网页来停止死循环
- 开发者一定要避免死循环的产生

### do...while循环

- do...while循环和while循环非常像，二者经常可以相互替代（不常用）
- 但是do..while的特点是不管条件成不成立，do循环体都会先执行一次

- 基本语法：
```js
do {
    // 循环体代码
} while (条件表达式);
```

使用场景：（需要至少执行一次的操作）
- 用户交互场景（菜单，输入验证）
- 重试机制实现
- 初始化加载配置
- 数据批量处理（至少处理一次）
