### 程序的执行顺序

>[!tip] 在程序开发中，程序有三种不同的执行方式

- 顺序 - **自上而下**，顺序执行代码
- 分支 - 根据**条件判断**，决定执行代码的**分支**
- 循环 - 让 **特定代码 重复** 执行

### 代码块的理解

- 代码块是多行执行代码的集合，通过花括号`{}`放到了一起
- 在开发中，一行代码很难完成某一个特定的功能，我们会将这些代码放到一个代码块中
- 在JavaScript中，我们可以通过**流程控制语句**来决定如何执行一个代码块
	- 通过会通过一些**关键字**来告知**JS引擎**代码要如何被执行
	- 比如**分支语句**，**循环语句**对应的关键字等

### 分支结构

- 分支结构的代码就是让我们**根据条件**来**决定代码的执行**
- 分支结构也叫**判断结构**或**选择结构**
- 几乎所有的编程语言都有分支结构（C、C++、OC、Java、Python等）

> JavaScript中常见的分支结构有：
- if分支结构
- switch分支结构

### if分支语句

>if分支结构有三种

- 单分支结构
	- if...
- 多分枝结构
	- if...else...
	- if...else if...else...

#### 单分支结构
- 单分支语句：if
	- if(...)语句计算括号内的条件表达式，如果计算结果是true，就会执行对应的代码块
```js
// 如果条件成立，那么执行代码块
if(条件判断) {
	// 执行代码块
}
```
