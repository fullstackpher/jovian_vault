### 认识JavaScript变量
- JS的变量是存储数据的容器，它存放的数据都是可变的

### 变量的定义、规范

> [!info] 在JS中命名变量，包含两部分：

- 变量的声明：在JS中声明变量使用`var`关键字，ES6中使用let、const方式
- 变量的赋值：使用 = 给变量进行赋值

这个过程也可以分开操作：

```js
var name;
name = 'jovian';
```

同时声明多个变量：

```js
var name, age, sex;
name = "Alice";
age = 25;
sex = "Female"
console.log(name, age, sex);
```

>[!danger] 变量的命名规范
>	变量的命名规则：必须遵守

- 首字母必须是一个字母、下划线（ _ ）或一个美元符号（ $ ）
- 其他字符可以是字母、下划线、美元符或数字
- 不能使用关键字或保留字命名
	- [关键字/保留字](https://developer.mozilla.org/zh-CN/docs/Web/JavaScript/Reference/Lexical_grammar)
- 变量**严格区分大小写**
- 多个单词使用**驼峰标识**
- 变量应该做到见名知意

>[!danger] 变量的使用注意

- 注意一：如果一个变量未声明就直接使用，那么会报错
