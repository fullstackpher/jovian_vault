### 认识JavaScript变量
- JS的变量是存储数据的容器，它存放的数据都是可变的

### 变量的定义、规范

> [!info] 在JS中命名变量，包含两部分：

- 变量的声明：在JS中声明变量使用`var`关键字，ES6中使用let、const方式
- 变量的赋值：使用 = 给变量进行赋值

这个过程也可以分开操作：

```js
var name;
name = 'jovian';
```

同时声明多个变量：

```js
var name, age, sex;
name = "Alice";
age = 25;
sex = "Female"
console.log(name, age, sex);
```

>[!danger] 变量的命名规范
>	变量的命名规则：必须遵守

- 首字母必须是一个字母、下划线（ _ ）或一个美元符号（ $ ）
- 其他字符可以是字母、下划线、美元符或数字
- 不能使用**关键字**或**保留字**命名
	- [关键字/保留字](https://developer.mozilla.org/zh-CN/docs/Web/JavaScript/Reference/Lexical_grammar)
- 变量**严格区分大小写**
- 多个单词使用**驼峰标识**
- 变量应该做到**见名知意**

>[!danger] 变量的使用注意

- 注意一：如果一个变量未声明（declaration）就直接使用，那么会报错
	- **Uncaught ReferenceError: message is not defined**
	- `name`是全局属性，直接输出name可能为空
- 注意二：如果一个变量有声明，但没有赋值，那么默认就是undefined
- 注意三：如果没有使用var声明变量，但不推荐这种定义方式（事实上会被加到window对象上）

### 数据类型和typeof

>[!info] JavaScript中的值都具有特定的类型的
-

- 我们将值赋给一个变量，这个变量就具备了特定的类型
- 一个变量前一刻是个字符串，下一刻就变成了一个数字
- 允许这种操作的编程语言，例如JavaScript，被称为“动态类型”的编程语言

>[!tip] JS中有8种基本的数据类型

- Number
- String
- Boolean
- Undefined
- Null
- Object
- BigInt
- Symbol

>[!info] typeof操作符

- 因为ECMAScript的类型系统是松散的，所以需要一种手段来确定任意变量的数据类型
	- typeof操作费就是为此而生的
- 对一个值使用typeof操作符会返回如下字符串之一：
	- "undefined"表示值未定义
	- "boolean"表示值为布尔值
	- "string"表示值为字符串
	- "number"表示值为数值
	- "object"表示值为对象（而不是函数）或null
	- "function"表示值为函数
	- "symbol"表示值为符号
