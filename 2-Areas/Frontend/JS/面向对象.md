在JS中一切皆对象，数组，函数都是对象的一种

### 认识对象

对象是 **键值对** 的无序集合，表示属性和值的映射关系

- Object类型的数据由属性组成，属性由属性名和属性值组成
- 属性值可以是任意类型的数据
- 属性名符合标识符命名规则，则省略引号
- 属性为函数的话，该属性可以被称为方法

语法：
- key和value之间用冒号分隔，每组key-value之间用逗号分隔，最后一组省略逗号


### 声明Object对象

#### 直接量方式

```js
var user = {
	username: 'Jovian',
	age: 28,
	'home-address' : '上海',
	schollAddress: '北京',
	hobbies: ['游戏', '唱歌'],
	child: {
		name: 'Jee',
		age: 3
	},
	getInfo: function() {
		console.log('obj2 getInfo');
	},
	eat: function() {
		console.log('jee爱吃');
	}
}
```

#### Object函数

```js
var obj = Object();
```

#### Object构造函数

```js
var obj = new Object();
```

### Object对象属性的读写

##### ①语法

> 属性访问

```js
对象名.属性名
```

>[!danger] 注意事项：访问不存在的属性和方法
>访问不存在的属性会返回：undefined
>访问不存在的方法会报错：Uncaught TypeError: user.drink is not a function

> 属性修改

```js
对象名.属性 = value
```

#### ②什么情况下必须使用 `[]`方式读写属性
