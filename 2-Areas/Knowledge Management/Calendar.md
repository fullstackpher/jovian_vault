# Obsidian Calendar 插件使用指南

## 插件概述

Calendar 是 Obsidian 的核心插件，提供可视化的日历视图，用于：
- 快速导航到任意日期的每日笔记
- 可视化笔记创建时间分布
- 管理每日例行事务

---

## 核心功能

### 1. 日历视图
- **月视图**：显示当前月份的日历
- **周视图**：显示当前周的详细视图
- 左侧边栏显示月份导航箭头
- 支持点击任意日期跳转

### 2. 每日笔记关联
- 点击日期可打开/创建对应的每日笔记
- 默认笔记命名格式：`yyyy-MM-DD`
- 支持自定义笔记模板

### 3. 热图显示
- 根据笔记数量显示深浅不同的背景色
- 颜色深度表示活跃程度
- 可快速识别高产日期

### 4. 每周摘要（Week Vault）
- 在周视图中显示该周创建的笔记数量
- 悬停查看详细信息

---

## 配置选项

在 **设置 → Calendar** 中可配置：

| 选项 | 说明 | 默认值 |
|------|------|--------|
| Weekly note | 设置每周摘要笔记模板 | - |
| Week start | 一周开始日期 | Sunday |
| Show week number | 显示周数 | 开启 |
| Show week vault | 显示每周笔记统计 | 开启 |
| Hotkey | 打开日历快捷键 | Ctrl/Cmd + Shift + K |

### 高级配置

```yaml
# 在笔记 frontmatter 中设置
---
calendar:
  start-of-week: monday    # 设置周一为每周开始
  week-note-template: "templates/weekly"  # 周记模板路径
---
```

## 使用技巧

### 快速跳转到今天
1. 使用快捷键 `Ctrl/Cmd + Shift + K`
2. 点击日历右下角的 "Today" 按钮

### 创建每日笔记
1. 在日历中点击任意日期
2. 自动创建并打开对应日期的每日笔记
3. 使用模板自动填充内容

### 追踪习惯/记录
1. 在每日笔记中记录习惯完成情况
2. 通过热图可视化坚持程度
3. 颜色越深 = 记录越频繁

### 周记管理
1. 配置 Weekly note 模板
2. 每周自动生成周总结
3. 包含该周所有笔记的链接

## 常见用例

### 用例 1：每日打卡
```
1. 每天打开 Obsidian
2. Calendar 中点击今天的日期
3. 在每日笔记中打勾完成的事项
4. 热图显示打卡记录
```

### 用例 2：项目进度追踪
```
1. 为每个项目创建独立文件夹
2. 在对应笔记中使用日期标签
3. Calendar 热图显示项目活跃度
```

### 用例 3：学习笔记管理
```
1. 每天学习后创建一条笔记
2. 使用统一的命名格式
3. Calendar 查看学习频率
```

- **Heatmap Calendar**：更详细的热图统计

## 快捷键速查

| 操作 | Windows/Linux | Mac |
|------|---------------|-----|
| 打开日历 | Ctrl + Shift + K | Cmd + Shift + K |
| 上个月 | ← 或 Page Up | 同左 |
| 下个月 | → 或 Page Down | 同左 |
| 回到今天 | Today 按钮 | 同左 |

---

## 注意事项

1. Calendar 插件需要 **Daily Notes** 插件启用才能正常工作
2. 热图颜色可在外观设置中自定义
3. 大量笔记时热图渲染可能略有延迟
4. 建议配合模板使用效果最佳
