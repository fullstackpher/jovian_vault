# 7.Daily 日周记 (Daily Notes)

> 每日记录、周记、月度复盘、习惯追踪

## 📍 定位

日周记文件夹是**时间维度的记录系统**，用于追踪每日学习、工作和生活。通过日复盘积累数据，周总结提炼要点，月度复盘审视方向。这是建立个人成长轨迹的核心区域。

**核心原则**：每日记录，定期回顾，持续改进。

## 📂 存储内容类型

- **每日日志**：每日学习内容、完成事项、心得体会
- **周记**：每周总结、进度回顾、下周计划
- **月度复盘**：月度目标回顾、成果评估、方向调整
- **季度总结**：季度大复盘、里程碑回顾
- **年度回顾**：年度总结、成长轨迹分析
- **习惯追踪**：习惯打卡记录、习惯养成数据
- **灵感记录**：每日产生的想法和创意

## 📏 命名规范

### 每日笔记命名

```
YYYY-MM-DD.md
```

### 周记命名

```
YYYY-Www-周记.md        # ww 为周数，如 2026-W05-周记.md
```

### 月度复盘命名

```
YYYY-MM-月度复盘.md      # 如 2026-01-月度复盘.md
```

### 示例

```
# 每日日志
2026-01-30.md
2026-01-29.md
2026-01-28.md

# 学习日报
2026-01-30 学习日报.md
2026-01-29 学习日报.md

# 周记
2026-W05-周记.md
2026-W04-周记.md

# 月度复盘
2026-01-月度复盘.md
2025-12-月度复盘.md
```

### 命名要求

| 规则 | 说明 |
|------|------|
| 日期格式 | 使用 ISO 格式 `YYYY-MM-DD` |
| 周数格式 | 使用 `YYYY-Www` 格式 |
| 分隔符 | 使用 `-` 连接各部分 |
| 避免字符 | 不使用 `\/:*?"<>|` |

## 🔄 记录周期

| 周期 | 动作 |
|------|------|
| 每日 | 填写每日日志/学习日报，记录当日学习 |
| 每周 | 撰写周记，回顾本周进展，规划下周 |
| 每月 | 完成月度复盘，评估目标达成情况 |
| 每季 | 季度大复盘，审视长期方向 |

## 🔗 与其他目录的连接

- **项目**：记录项目进展、遇到的问题、解决方案
- **领域**：将学习心得提炼到对应领域
- **收件箱**：捕获每日产生的灵感和想法
- **资源**：记录学习过程中发现的好资源

## ⚠️ 注意事项

1. **不要**追求完美记录，先完成再完美
2. **不要**在单篇日志中写过多内容，保持简洁
3. **保持**格式一致，便于后续统计和分析
4. **定期**回顾历史记录，发现成长轨迹
5. **结合**Dataview进行数据统计和可视化

## 💡 使用技巧

### 使用 Dataview 统计学习时长

```dataview
TABLE file.name, 记录日期 AS "日期"
FROM "7.Daily 日周记/学习日报"
WHERE file.cday >= date("2026-01-01")
SORT file.cday DESC
```

### 建立每日回顾习惯

每天睡前花 5 分钟完成当日记录，包括：
- 今日完成事项
- 明日优先事项
- 今日小确幸

### 周记写作要点

- 本周完成的核心任务
- 本周学到的关键知识点
- 本周遇到的挑战和解决方案
- 下周的重点目标和计划

## 📊 复盘框架

### 复盘四步法

1. **回顾目标**：当初设定的目标是什么
2. **评估结果**：实际完成情况如何
3. **分析原因**：成功/失败的原因是什么
4. **总结经验**：下次如何做得更好

### 月度复盘问题

- 本月最重要的成果是什么？
- 本月最大的挑战是什么？
- 本月学到了什么新技能？
- 下个月最重要的目标是什么？
- 需要调整或改进的地方？
